# 🛡️ TJ Security AI Bot

Телеграм-бот для проверки ссылок, файлов, AI-обработки фото и многого другого.

## 🔥 Возможности

- 🔗 Проверка ссылок (WHOIS, SSL, редиректы, AbuseIPDB)
- 📁 Проверка файлов (MD5/SHA256, расширения)
- 🤖 AI-помощник на базе GPT-4o-mini
- 🖼 AI-обработка фото (удаление фона, стилизация)
- 👤 Система баллов и тарифов
- 👥 Реферальная программа (+2 балла)
- 👑 Админ-панель
